Instalando o GIT no Windows

  • 1. Faça o login em sua conta https://github.com/
  • 2. Baixar o GIT para Windows e Instale
  • 3. Na sua área de trabalho crie uma nova pasta com nome: testegit, e coloque um arquivo .html.
  • 4. No seu terminal (CMD) encontre a pasta parainicie o git:
  • git init
  • git add *
  • git commit "meu primeiro git commit"
  • git remote add origin https://github.com/seu_usuario_git/testegit.git
  • git push --set-upstream origin master
